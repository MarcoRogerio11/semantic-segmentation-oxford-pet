# Segmentação Semântica com U-Net

## Objetivo

Implementar um modelo de segmentação semântica utilizando Deep Learning para segmentar animais do conjunto de dados Oxford-IIIT Pet.

## Dataset

Oxford-IIIT Pet Dataset.

## Modelo

U-Net implementada em PyTorch.

## Resultados

- Accuracy: 71.9%
- IoU: 64.9%
